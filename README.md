🚀 Sobre o Projeto
Este é um projeto de uma Pokedex digital, criado como parte dos estudos de desenvolvimento web. A aplicação consome a PokéAPI para buscar e exibir uma lista de Pokémon, permitindo a visualização de seus detalhes de forma dinâmica e com um design responsivo.

✨ Veja o projeto ao vivo!
💻 Tecnologias Utilizadas
O projeto foi construído utilizando as seguintes tecnologias:

HTML5: Estrutura semântica e organizada.

CSS3: Estilização moderna com Flexbox, Grid Layout e Media Queries para responsividade.

JavaScript (ES6+): Manipulação da DOM, consumo de API com fetch e lógica de programação assíncrona.

✅ Funcionalidades
[x] Listagem de Pokémon: Exibe uma lista de Pokémon obtida diretamente da API.

[x] Paginação: Botão "Load More" que carrega mais Pokémon na lista de forma dinâmica.

[x] Estilização Dinâmica: Os cards dos Pokémon mudam de cor de acordo com o seu tipo principal.

[x] Design Responsivo: A interface se adapta perfeitamente a diferentes tamanhos de tela, de celulares a desktops.

[x] Efeitos Visuais: Animações e efeitos de hover para uma experiência de usuário mais agradável e interativa.
