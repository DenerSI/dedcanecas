# 📚 Pokédex: Uma Nova Versão com React e Vite!
<br>

### Este projeto Pokédex foi construído do zero, usando a modernidade e a velocidade do React e do Vite. O objetivo é oferecer uma experiência de usuário fluida e informativa, com um design otimizado e um código limpo.

<br>

# ✨ Destaques e Funcionalidades
* ### Modal de Detalhes Interativo: Ao clicar em um Pokémon, um modal responsivo exibe informações detalhadas.

* ### Visuais Dinâmicos: O cabeçalho do modal se adapta com a cor do tipo principal do Pokémon, criando uma identidade visual única.

* ### Base Stats com Barras de Progresso: Uma visualização clara dos atributos de combate, incluindo um total geral para facilitar a comparação.

* ### Informações Abrangentes: Detalhes como altura, peso, habilidades, espécie e porcentagens de gênero estão disponíveis.

* ### Código Otimizado: O projeto foi completamente refatorado (HTML, CSS, JavaScript) para garantir uma estrutura padronizada, concisa e de fácil manutenção.

* ### Responsividade Aprimorada: A aplicação e o modal de detalhes se adaptam perfeitamente a diferentes tamanhos de tela, proporcionando uma experiência consistente em dispositivos móveis e desktops.

### 🛠️ Instalação e Uso

### Para rodar este projeto localmente, siga os passos abaixo:

### 1.  Clone o repositório para o seu computador:
    ```bash
    git clone (https://github.com/user/seu-repositorio.git)
    ```
### 2.  Entre na pasta do projeto:
    ```bash
    cd seu-repositorio
    ```
### 3.  Instale as dependências:
    ```bash
    npm install
    ```
### 4.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
