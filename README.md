# Pokédex

Este projeto é uma Pokédex interativa, que exibe uma lista de Pokémon e permite carregar mais itens conforme o usuário interage com a página.

## Funcionalidades

Exibe uma lista de Pokémon consumindo dados de uma API (PokéAPI).

Possui um botão "Load More" para carregar mais Pokémon.

Layout simples e organizado com HTML e CSS.

Código modular com separação de arquivos (HTML, CSS e JavaScript).
