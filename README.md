# 🌟 Pokédex DIO - Em Construção

🚧 **Atenção:** Este projeto ainda está em desenvolvimento. O objetivo é criar uma Pokédex incrível consumindo uma API REST do mundo Pokémon, aplicando todos os fundamentos de desenvolvimento Web aprendidos na imersão da DIO.

## Sobre o Desafio

O desafio consiste em:

- Entender a importância do Git/GitHub nos Desafios de Projeto da DIO.  
- Explorar o mundo Pokémon através do consumo de uma API REST.  
- Criar uma Pokédex funcional utilizando JavaScript, HTML e CSS.  

Este projeto serve para colocar em prática todos os conceitos aprendidos durante a imersão, incluindo manipulação do DOM, requisições HTTP, e versionamento de código.

## Tecnologias

- JavaScript  
- HTML  
- CSS  

## Observação

O projeto ainda está em desenvolvimento. Em breve, ele será atualizado com todas as funcionalidades da Pokédex.
